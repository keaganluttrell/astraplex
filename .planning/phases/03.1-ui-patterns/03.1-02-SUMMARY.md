## Decisions Made
- LiveViews call shell functions directly in render/1 (not via router layout option) for explicit layout control
- layout: false on live_sessions prevents Phoenix from wrapping content in a default app layout
- Used daisyUI CSS `dock` class directly since DaisyUIComponents has no dock component
- Used DaisyUIComponents modal with `modal-bottom` class and `close_on_click_away` for apps bottom sheet
